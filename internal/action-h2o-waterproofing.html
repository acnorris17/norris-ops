Pre-Closing Required Action — Do Not Skip
Call H2O Waterproofing — Get Written Confirmation Before Closing

Action Required: Contact H2O Waterproofing directly and obtain written documentation regarding any waterproofing work, warranties, or assessments related to the property before proceeding to closing.

  • Call H2O Waterproofing to discuss the property and any work performed, scheduled, or recommended
  • Request a written statement or report confirming the scope of work, warranty coverage, and current condition
  • Obtain the written documentation via email or letter — verbal confirmation alone is not sufficient
  • Review the written confirmation and verify it addresses all waterproofing concerns before closing
  • Provide a copy of the written confirmation to the closing attorney or title company for the file
Timing: Must be completed before closing date — no exceptions

Why This Matters

Waterproofing issues discovered after closing become the buyer's responsibility. Getting H2O Waterproofing's written confirmation on record protects against undisclosed conditions and ensures any existing warranties or guarantees transfer properly.

A written statement from H2O Waterproofing should cover: work previously performed, active warranty details, and any known conditions. This document becomes part of the permanent closing file.
