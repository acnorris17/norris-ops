What's Not Real / MisleadingClaims that overstate current capability — honest assessment

  • ✗
    "AI answers your phone and handles customer calls"
    There is no voice AI answering calls for Norris Utilities®. The agent is text-based inside Claude Projects. Phone calls still go to Aaron directly at 205-500-1343. Voice AI integration would require a separate platform like Bland.ai or Vapi, plus significant setup and testing.
  • ✗
    "Automated email sending and follow-up sequences"
    The agent composes emails in Aaron's voice — but it does not send them automatically. Every email requires Aaron to copy, paste, review, and hit send manually. True automated sending would require integration with an email API (SendGrid, Gmail API) and a scheduling system.
  • ✗
    "Real-time data sync with suppliers and customers"
    The agent has no live connections to external systems. It cannot pull real-time pricing from Samson Rope, check Skylift inventory, or see Bay Shore delivery schedules. All data is static, loaded through the context document, and must be manually updated when prices or contacts change.
