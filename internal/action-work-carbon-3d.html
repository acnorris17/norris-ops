Action Item — Software Update

Update Carbon 3D

Captured: April 13, 2026  |  Priority: Active — Update Required

⚙ Carbon 3D software update needed — ensure platform and project files stay current

Task Summary

Update the Carbon 3D software platform to the latest version. Ensure all project files, printer profiles, and design assets are current and compatible with the updated environment. Back up existing work before proceeding.

Update Checklist

  • Check current Carbon 3D version and identify available updates
  • Back up all existing project files and custom printer profiles before updating
  • Download and install the latest Carbon 3D software release
  • Verify all existing designs and print settings remain functional after the update
  • Test a sample print to confirm quality and calibration are unaffected
  • Document the updated version number and any changes to workflows

Focus Areas for Update

Backup First

Copy all current project files, resin profiles, and custom settings to a backup location before starting the update. This protects against any compatibility issues with the new version.

Materials Library

Review the updated materials library for any new resins or updated material profiles. Check if any current material settings need recalibration with the new software version.

Software Features

Review release notes for new features, bug fixes, and workflow improvements. Identify any changes that affect current Norris Utilities® part designs or production processes.

Validation Testing

After the update, run a test print on an existing part design to confirm print quality, dimensional accuracy, and surface finish match pre-update results.

Priority Notes

  • Keeping Carbon 3D current ensures access to the latest materials library and improved print accuracy
  • Updated software maintains eligibility for Carbon 3D technical support
  • Schedule the update during a window when the printer is not running active production jobs
  • If the update includes firmware changes, allow extra time for printer recalibration

Next Steps

1
Back up all project files. Save current designs, printer profiles, and resin settings to a separate folder before touching the software.
2
Check for the latest version. Log into the Carbon 3D dashboard and check for pending software and firmware updates.
3
Install the update. Follow Carbon 3D’s update process. Apply any firmware updates to the printer hardware as prompted.
4
Verify and test. Open existing project files to confirm compatibility. Run a test print to validate quality and calibration.
5
Document the update. Record the new version number, any workflow changes, and the results of the test print for future reference.
