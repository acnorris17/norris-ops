Custom Domain Setup

Connect ops.norrisutilities.com to this portal

1DNS Records (at your domain registrar)

Add this record for ops.norrisutilities.com:

TypeHost / NameValue / TargetTTL
CNAMEopsacnorris17.github.io3600

2GitHub Pages Settings

  1. Go to github.com/acnorris17/norris-ops → Settings → Pages
  2. Under Custom domain, enter ops.norrisutilities.com
  3. Click Save
  4. Check Enforce HTTPS (may take a few minutes to become available)

3Verify

After DNS propagation (usually 5–30 minutes):

  • ✓ https://ops.norrisutilities.com loads the portal
  • ✓ HTTPS padlock shows in browser
  • ✓ acnorris17.github.io/norris-ops/ redirects to custom domain
Note: The CNAME file in the repo root tells GitHub Pages which custom domain to use. Do not delete it or the custom domain will disconnect.
